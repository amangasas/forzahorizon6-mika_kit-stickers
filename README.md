# forzahorizon6-mika_kit-stickers

24 custom Mika Kit vinyl sticker designs for Forza Horizon 6.

Made with Forza Painter v1.6.0 because apparently my GPU wasn't suffering enough already.

## Render Settings

| Version | Preset |
|---|---|
| 1000 Objects | `"i hate my gpu"` |
| 3000 Objects | `"best quality"` |
